## 🔄 How the Pipeline Will Use Categories

When an article URL is provided:

1. **Article fetcher** extracts the category from the article HTML
2. **Script generator** includes category in the Bedrock prompt
3. **Video compositor** selects the prestitched video based on matching tags
4. **Result:** Category-appropriate background video for each article
